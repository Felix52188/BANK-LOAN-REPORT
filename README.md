# BANK-LOAN-REPORT

## Project Overview
This project analyzes a publicly available HR performance dataset using PowerBI  to identify factors that influence HR performance and present insights through interactive dashboards.

## Tools Used
* PowerBI
* GitHub

## Dataset
* Generated from AI software

## Project Objectives
* Analyze bank loan.
* Explore the relationship between good loan and bad loan.
* Build interactive dashboards.
* Generate insights and recommendations.

## Dashboards
1. BANK-LOAN-REPORT | SUMMARY
2. BANK-LOAN-REPORT | OVERVIEW
3. BANK-LOAN-REPORT | DETAILS

## Key Insights
* Good loan percentage was established
* Bad loan was shown to be below 20%
* Total funded amount and total recieved amount was accertained.
* Loan by term duration was established
